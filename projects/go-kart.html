Custom Go-Kart

Build video — add MP4 or YouTube embed when ready

This is a from-scratch, single-seat kart powered by a Lifan 212 cc electric-start engine with a custom push-to-start switch. Power goes through a 30-series torque converter to a live axle. The frame is a 1×1×.120″ A513 square-tube steel chassis designed to be straightforward to build and maintain.

Engine Selection

We chose the Lifan 212 cc electric-start after comparing common 212-class options. Converting a pull-start engine to electric-start adds parts, cost, and risk (ring-gear flywheel, starter, wiring updates). The Lifan ships with electric-start and a harness that supports a clean push-button start, which fits the goal of a practical kart that’s easy to start, test, and maintain.

Chassis — Design

We moved from a round-tube idea to square-tube using 1×1×.120″ A513. Square tube avoids bending equipment, keeps cuts repeatable, and makes alignment and fixturing straightforward for a first build.

Chassis — Manufacturing

The chassis is fabricated in-house: cutting the steel, cleaning and beveling with an angle grinder, tack-up, and welding. The weld sequence is planned to control distortion; assembly continues as weld-out completes.

Powertrain & Drivetrain

The Lifan 212 cc drives a 30-series torque converter, then a chain to the live axle. The live axle uses hubs and lock collars to locate the sprocket, brake, and bearings along the shaft so alignment stays set in use.

Issue: The TAV backplate limited belt alignment with the available engine-mount slot travel.
Fix: Added spacers and a slotted secondary bracket to set center distance while keeping the TAV parallel.

Electrical — Push-to-Start

Start path: battery → main fuse → master switch → fused push-button branch → starter relay → starter. The kill switch grounds ignition. Grounds are tied to a single chassis point for reliability.

Issue: Starter latched once due to a shared ground path with the kill switch.
Fix: Added a dedicated ground strap to the starter case and separated returns; ferrule-crimped terminations.

Structural Analysis (FEA)

Quick structural checks (finite element analysis) were run to sanity-check deflection and stress at likely hot-spots before committing to steel: steering column mount, axle bearing tabs, and representative frame spans.
